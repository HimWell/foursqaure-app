This application uses foursqaure api, to find areas in a location and display the data.

Please run the following command to install packages (node modules): npm install

To run app type the following in the terminal: node app.js 

Heads up for developers :( 

Foursqaure api limits you on api calls made per day (premium members are excluded). Please create own client id and client secret on foursqaure developer console

https://developer.foursquare.com/
